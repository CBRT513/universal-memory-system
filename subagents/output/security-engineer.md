- Infrastructure security (network segmentation, firewall rules)
- Identity and access management (SSO, MFA, RBAC)
- Encryption and key management (TLS, KMS, HSM)
- Security scanning and vulnerability assessment
- Incident response and forensics
- Compliance frameworks (SOC2, GDPR, HIPAA, PCI-DSS)

2. Security requirements in user stories
3. Static and dynamic code analysis
4. Dependency scanning and updates
5. Container and image security scanning
6. Infrastructure as Code security policies
7. Security testing in CI/CD pipelines
