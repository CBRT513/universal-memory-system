💡 Pro Tips

  • Select before clicking: Select specific text for quick capture of key insights
  • Use status wisely: Mark stable solutions as "working" to protect them
  • Tag consistently: Use consistent tags to build searchable knowledge bases
  • Rate importance: High importance memories appear first in context generation
  • Capture failures: Remember what didn't work to avoid repeating mistakes

🔗 Integration with CLI

Memories captured through the browser are immediately available via CLI:

# Search memories captured from any AI platform memory search "React optimization" --source chatgpt # Get context including browser-captured memories memory context --relevant-to "database performance" --copy # View all memories from a specific AI platform memory search --tags claude --format markdown
