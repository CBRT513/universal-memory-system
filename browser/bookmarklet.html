🔖 UMS Capture Bookmarklet

Status: UMS API must be running on http://localhost:8091

Drag this button to your bookmarks bar:

📄 Capture to UMS

How to Install

  1. Make sure your bookmarks bar is visible (Ctrl+Shift+B on Windows/Linux, Cmd+Shift+B on Mac)
  2. Drag the blue "📄 Capture to UMS" button above to your bookmarks bar
  3. That's it! The bookmarklet is now installed

How to Use

  1. Navigate to any article or webpage you want to capture
  2. Click the "📄 Capture to UMS" bookmarklet in your bookmarks bar
  3. Wait for the success message (usually 1-3 seconds)
  4. The article is now saved in your UMS database!

When to Use This

Use the bookmarklet when:

  • ✅ The site blocks automated capture (paywalls, bot protection)
  • ✅ You're already reading an article and want to save it
  • ✅ The normal capture returns a 403 Forbidden or 401 Unauthorized
  • ✅ You want to capture content that requires login

⚠️ Requirements

  • UMS API must be running on http://localhost:8091
  • CORS must be enabled in the API (already configured)
  • The bookmarklet works best on article-style pages
  • Some sites with strict CSP policies may block the bookmarklet

Bookmarklet Source Code (for reference):

(async () => {
    // Load Readability library
    const script = document.createElement('script');
    script.src = 'https://cdn.jsdelivr.net/npm/@mozilla/readability@0.5.0/Readability.js';
    document.head.appendChild(script);
    await new Promise(resolve => script.onload = resolve);
    
    // Parse the page
    const documentClone = new DOMParser().parseFromString(
        document.documentElement.outerHTML, 
        'text/html'
    );
    const article = new Readability(documentClone).parse();
    
    if (!article) {
        alert('Could not extract article content');
        return;
    }
    
    // Prepare payload
    const payload = {
        url: location.href,
        title: article.title || document.title,
        text: article.textContent || document.body.innerText,
        project: 'vader-lab'
    };
    
    // Send to UMS
    try {
        const response = await fetch('http://localhost:8091/api/capture/from_client', {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify(payload)
        });
        
        if (response.ok) {
            const data = await response.json();
            alert(`✅ Captured to UMS!\nJob ID: ${data.job_id}`);
        } else {
            alert(`❌ Capture failed: ${response.status} ${response.statusText}`);
        }
    } catch (error) {
        alert(`❌ Error: ${error.message}\n\nMake sure UMS is running on http://localhost:8091`);
    }
})();

Alternative: Manual Installation

If drag-and-drop doesn't work:

  1. Right-click your bookmarks bar and select "Add page" or "Add bookmark"
  2. Name it: 📄 Capture to UMS
  3. For the URL, copy the entire code from the code block above (starting with javascript:)
  4. Save the bookmark
Test it: Try capturing this page itself by clicking the bookmarklet!
